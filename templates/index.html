How to Use

1. Edit config.yaml to configure your email sender, LLM API key, and user calendars.

2. For Google Calendar, ensure you have credentials.json in the project root and authenticate when prompted.

3. Run the standalone app: python calmind/main.py

4. Or run the web app: python calmind/webapp.py and navigate to http://127.0.0.1:5000/ in your browser.

5. Check the reports/ folder for generated HTML and Markdown files, and your configured email for the summary.
